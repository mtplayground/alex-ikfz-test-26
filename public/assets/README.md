# Placeholder Asset Pack

This repository includes a lightweight placeholder asset pack for early Phaser integration.

- `atlases/` contains sprite atlas PNGs with matching Phaser atlas JSON.
- `tilemaps/` contains Tiled-compatible JSON map data.
- `tilesets/` contains the PNG tilesheet used by the map JSON.
- `audio/` contains placeholder BGM and SFX WAV files.

The assets are intentionally simple and deterministic so later preload, animation, and collision issues can build on stable file names and frame coordinates.
